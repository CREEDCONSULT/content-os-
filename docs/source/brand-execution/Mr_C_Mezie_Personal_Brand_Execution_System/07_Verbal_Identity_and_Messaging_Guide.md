## Communication structures

### Principle-led
```text
Point → Why it matters → Example → Principle → Action
```

### Story-led
```text
Situation → Tension → Decision → Result → Lesson
```

### Educational
```text
What it is → Why it matters → How it works → Risks → Next step
```

### Market analysis
```text
Event → Mechanism → Possible impact → Risks → What to watch
```

## Tone by platform
- Instagram: human, direct, visual, motivating.
- LinkedIn: structured, credible, professional, reflective.
- X: concise, curious, sharp, provocative without outrage.
- YouTube: educational, detailed, calm, story-led.

## Avoid
Guaranteed, secret, easy money, everyone must, this cannot fail, overnight wealth, alpha posturing, empty hustle language.
